# Hemocam Web App

A specialized web application for hemocyte analysis and visualization, combining advanced parsing, web technologies, and Python-based processing.

## Project Overview

The Hemocam Web App is designed to provide a comprehensive platform for analyzing and visualizing hemocyte data with an intuitive web interface and powerful backend processing capabilities.

## Technology Stack

This project leverages a diverse technology stack:

- **Yacc (54.6%)** - Parser generation for complex data parsing and language processing
- **HTML (27.5%)** - Frontend markup and web interface
- **Python (11.5%)** - Backend logic, data processing, and business logic
- **Lex (6.4%)** - Lexical analysis for tokenization and pattern matching

## Features

- 📊 **Data Analysis** - Advanced parsing and processing of hemocyte data
- 🎨 **Web Interface** - Responsive HTML-based user interface
- 🔧 **Backend Processing** - Python-powered data processing and analysis
- ⚙️ **Lexical Analysis** - Efficient tokenization and pattern recognition

- Python 3.x
- Web server or development environment
- Yacc/Lex tools (for parser/lexer compilation)

### Installation

1. Clone the repository
```bash
git clone https://github.com/Laboni108/Hemocam_web_APP.git
cd Hemocam_web_APP
```

2. Install Python dependencies
```bash
pip install -r requirements.txt
```

